# Daily Capture

## Overview

Daily Capture is a web app that allows users to share photos of the same place every day.

## Features

- Share photos of the same place every day
- View photos of the same place every day
- View photos of the same place every day

## Tech Stack

- Next.js
- Tailwind CSS
- Supabase
- Shadcn UI
- TypeScript

## Development

```bash
pnpm dev
```

## Deployment

```bash
pnpm build
```
